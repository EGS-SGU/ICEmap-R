ICEMAP-R

Ice Mapping Automated Procedure from Radar data

 

Description
IceMAP-R© (Ice Mapping Automated Procedure from Radar data) is designed to process radar satellite images in C-Band and HH polarization, to automatically produce a map of the different types of ice on a river, at the time of image acquisition.

Parameter Details

INFILE

Name of the file containing the ortorectified HH radar image

INFILEC

In INFILE, channel number containing the HH radar image to use

INMASK

Name of the file containing the georeferenced vector of the river

INMASKS

In INMASK, Segment number containing the vector to use

INPCT

Name of the file containing the pseudo-color table corresponding to the legend to use OR name of the file containing a custom pseudo-color table (.txt).

Predefined pseudo-color tables (legends) are used for the freeze-up (freeze) and breakup (thaw) period. Custom pseudo-color tables must contains values from 0 to 255, but only values from 1 to 9 are used (corresponds to the 9 classes of the output).

OUTFILE

Name of the output file

The output file will be in Tagged Image File Format (.TIF)

LOGFILE

Name of the log file

If no log file is provided, the log details are only display on terminal.

LOGDETAILS

Threshold of log details

From 1 to 5. 1=high detail log, 5=low detail log, default=2. Details levels are related to the handler level of the python library logging.

Example

River ice mapping will be done on the file “radar_HH.pix” containing the radar C-HH image in channel “1”. The vector of the river is contained in segment 1 of the file “ river_mask.shp”. The legend to use is the pseudo-color table “freeze”. The resulting map will be saved in file “ icemapr_results.tif”. All operational details of the process will be written to the file icemapr.log using level 2 logger.

 

infile     = "radar_HH.pix"   # input file

infilec    = 1                # channel of the HH image

inmask     = "river_mask.shp" # river polygon file

inmasks    = 1                # segment number for the polygon

inpct      = "freeze"         # pct to use for the legend

outfile    = "icemapr_results.tif"   # output file

logfile    = "icemapr.log"    # log file

logdetails = 2                # set the logger to info level

 

icemapr(infile, infilec, inmask, inmasks, inpct, outfile, logfile, logdetails)
